Creating a List

A list is defined using square brackets [], and elements are separated by commas. 

# Example: Creating a list of integers
numbers = [1, 2, 3, 4, 5]

# Example: Creating a list of mixed data types
mixed_list = [1, "Hello", 3.14, True]

Accesing List Elements

List elements are accessed using zero-based indexing. You can use positive indices to access elements from the start or negative indices to access elements from the end. 

# Example: Accessing elements
numbers = [10, 20, 30, 40]

# Access the first element
print(numbers[0])  # Output: 10

# Access the last element
print(numbers[-1])  # Output: 40

Modifying List

Lists are mutable, meaning their elements can be changed or updated after creation. 

# Example: Modifying a list
numbers = [10, 20, 30]
numbers[1] = 25  # Change the second element
print(numbers)  # Output: [10, 25, 30]

List Methods

Python provides several built-in methods to manipulate lists:

1. append(): Adds an item to the end of the list.

numbers = [1, 2, 3]
numbers.append(4)
print(numbers)  # Output: [1, 2, 3, 4]

2. insert(): Inserts an item at a specified index.

numbers.insert(1, 10)
print(numbers)  # Output: [1, 10, 2, 3, 4]

3. remove(): Removes the first occurrence of an item.

numbers.remove(10)
print(numbers)  # Output: [1, 2, 3, 4]

4. pop(): Removes and returns the item at a specified index. If no index is provided, it removes the last item.

numbers.pop()
print(numbers)  # Output: [1, 2, 3]

5. sort(): Sorts the list in ascending order (can also sort in descending order).

numbers.sort()
print(numbers)  # Output: [1, 2, 3]

# Sorting in descending order
numbers.sort(reverse=True)
print(numbers)  # Output: [3, 2, 1]

6. reverse(): Reverses the elements of the list.

numbers.reverse()
print(numbers)  # Output: [1, 2, 3]

7. clear(): Removes all elements from the list.

numbers.clear()
print(numbers)  # Output: []

List Comprehension

List comprehension offers a concise way to create lists by applying an expression to each item in an iterable. 

# Example: Create a list of squares of numbers
squares = [x**2 for x in range(5)]
print(squares)  # Output: [0, 1, 4, 9, 16]

List Slicing

Slicing allows you to get a sublist from a list by specifying a start and end index. 

numbers = [10, 20, 30, 40, 50]

# Get the first three elements
sublist = numbers[:3]
print(sublist)  # Output: [10, 20, 30]

# Get the last two elements
sublist = numbers[-2:]
print(sublist)  # Output: [40, 50]

Iterating Through a List

You can iterate through lists using loops, which allows you to access each element. 

# Example: Iterating with a for loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
# Output:
# apple
# banana
# cherry

Nested Lists

A list can contain other lists, allowing for the creation of multi-dimensional arrays (e.g., matrices). 

# Example: 2D list (Matrix)
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(matrix[0])  # Output: [1, 2, 3]
print(matrix[0][1])  # Output: 2

Common List Operations

1. len(): Returns the number of items in the list.

numbers = [1, 2, 3]
print(len(numbers))  # Output: 3

2. in: Checks if an item is in the list.

print(2 in numbers)  # Output: True

3. count(): Returns the number of occurrences of an item in the list.

numbers = [1, 2, 2, 3]
print(numbers.count(2))  # Output: 2
